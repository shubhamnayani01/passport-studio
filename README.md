# Passport Photo Studio

Professional AI-assisted passport photo generator built for local studios, print shops, and cyber cafés.

Generate passport and visa photo sheets with:
- automatic face detection
- smart crop adjustment
- AI enhancement tools
- multiple country presets
- print-ready sheet generation
- mobile-friendly uploads
- HEIC/HEIF support
- printer alignment controls

---

## Features

### Smart Upload System
- JPG / PNG / WEBP / HEIC / HEIF support
- Mobile-friendly upload pipeline
- Automatic image optimization
- Large image downscaling
- Memory-safe processing

### AI Enhancement Tools
- Brightness
- Contrast
- Saturation
- Sharpness
- Warmth
- Skin smoothing
- One-click enhancement presets

### Passport Utilities
- Auto face detection
- Smart crop framing
- Manual crop adjustment
- Country presets
- Multiple print sheet sizes
- Background color selection

### Print Support
- A4 / 4x6 / Letter layouts
- DPI selection
- Manual printer alignment
- Print-ready export
- PNG download support

### Studio Workflow Features
- Fast processing
- Simple UI
- Local studio oriented
- Browser-based workflow
- No software installation required
